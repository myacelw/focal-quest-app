## 1. 目标与背景

现在 `TrainingPage` 视标按 `optotypeHeightPx(acuity, pxPerMm)` 从视力级别换算物理尺寸——0.8 级别 ≈ 0.73mm，iPad 上只有几像素，孩子看不清，训练没法真正用。

**关键认知**：翻转拍训练的难度来自 **±2D 镜片强迫的调节**，视标只要"能看清但要努力"即可，本就该**比视力测量阈值大**。所以放大合理，不违背医学，只需保留一个"等效视力级别"锚点作参考。

**目标**：让家长/孩子在准备页用滑块把视标调到合适大小（带实时预览），训练能真正投入使用。

## 2. 范围

**做**
- 准备页把"视力级别"下拉换成"视标大小"滑块（物理毫米）+ 实时预览 E + 等效视力级别标注
- 训练视标按选定毫米 × 标定 px/mm 显示
- 大小存 localStorage，每次训练前可微调
- 加 `acuityFromHeightMm` 反函数算等效视力级别

**不做**
- 不引入新数据表、不改统计/勋章逻辑、不动其他页面
- 不做摄像头测距、不改 40cm 距离假设

## 3. 交互（准备页）

- **"视标大小"滑块**：范围 **2–20mm**，默认 **6mm**，步进 0.5mm。
- 滑块旁**实时预览**：一个按 `sizeMm × pxPerMm` 显示的 E（真实物理大小），孩子看着调到"能看清但要努力"。
- 预览下方标注 **等效视力级别**（如"≈ 0.10 级别"）作医学参考。
- 若未标定（`fzp.cssPxPerMm` 缺失）：提示先去标定（沿用现有逻辑）。
- 大小存 localStorage key **`fzp.optotypeSizeMm`**，默认取上次值。

## 4. 内部逻辑

- 训练视标像素：`heightPx = sizeMm × pxPerMm`（直接用毫米，不再从 acuity 换算）。
- `optotype-size.ts` 加反函数（纯函数）：
  - `acuityFromHeightMm(heightMm, distanceMm=400): number`
  - 由 `h = 2·D·tan(rad/2)` 反解：`rad = 2·atan(h/(2D))`；`arcMin = rad·(180·60)/π`；`acuity = 5/arcMin`
  - 用途：① 准备页显示等效视力级别；② `saveSession` 时把 `SessionRow.acuity` 存为等效视力级别 → **统计/勋章的 acuity 语义不变，不改 DB schema**。
- 往返一致：`acuityFromHeightMm(optotypeHeightMm(a)) ≈ a`。

## 5. 改动范围（文件）

- `src/training/optotype-size.ts`：加 `acuityFromHeightMm` + 测试。
- `src/training/TrainingPage.tsx`：
  - 准备页：`acuity` select → `sizeMm` 滑块（从 localStorage 读默认）+ 预览 `<TumblingE>` + 等效级别标注。
  - `state`：`acuity` → `sizeMm`（存 localStorage）。
  - 训练视标 `heightPx = sizeMm × pxPerMm`。
  - `saveSession`：`acuity: acuityFromHeightMm(sizeMm)`。
- 复用：`TumblingE`（预览用）、`optotypeHeightMm`（反函数往返测试）。

## 6. 测试策略

- **纯逻辑 TDD**：`acuityFromHeightMm`——已知算例（如 heightMm≈0.582 → acuity≈1.0）、往返（`acuityFromHeightMm(optotypeHeightMm(a))≈a`，a 取 0.3/0.8/1.0）、随尺寸增大等效级别减小。
- **手动验证**：准备页滑块调大小、预览 E 实时变化且物理尺寸合理（尺子量）、等效级别标注更新、训练视标用新大小、刷新记住上次大小、结算/统计正常。

## 7. 默认决策（已与用户确认）

1. 直接调"视标大小"（毫米）+ 实时预览，不用"视力级别+倍数"两旋钮
2. 范围 2–20mm、默认 6mm、步进 0.5mm（具体值家长现场按预览定）
3. 保留"等效视力级别"标注作医学锚点
4. `SessionRow.acuity` 存等效视力级别，不改 DB schema
5. 大小存 localStorage，每次训练前可微调

## 8. 非目标与后续

- 后续：iPad 真机复验（标定物理尺寸、孩子实际舒适大小、发音识别率）。
- 迭代 2 其余：趣味关卡皮肤、彩蛋、积分兑换、限时挑战。
