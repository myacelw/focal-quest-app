## 1. 目标

把 1b 已落库的训练记录（`sessions` 表）聚合成家长看得懂的图表：训练次数、时长、正确率、CPM 走势，按日/周/月查看。让家长直观看到孩子练了多少、调节力（CPM）有没有进步。

**成功标准**：统计页能按日/周/月切换，正确聚合出各期指标，用折线/柱状图直观展示；无训练数据时有友好空态。

## 2. 范围边界

**做（1c）**
- 「统计」页，日/周/月维度切换
- 汇总卡片：累计训练次数、累计时长
- CPM 走势折线图、正确率走势折线图、训练次数柱状图
- 从 `sessions` 聚合（纯函数）
- 手绘 SVG 图表组件（无第三方图表库）

**不做（推迟）**
- 智能趋势分析 / 一元回归 / 训练处方 → 迭代 3+
- 数据导出、云同步 → 迭代 3+
- 图表交互（tooltip/缩放）→ 以后按需
- checkins 的积分/streak 历史图表 → 以后（1c 聚焦 sessions 训练指标）

## 3. 展示内容

- **汇总卡片**（页顶）：累计训练次数（sessions 条数）、累计训练时长（sum elapsedSec，显示为分钟）
- **CPM 走势**折线图：每期平均 CPM（核心，反映调节力变化）
- **正确率走势**折线图：每期平均正确率（%）
- **训练次数**柱状图：每期训练节数

## 4. 维度切换（日 / 周 / 月）

- **日**：按 `date`（YYYY-MM-DD）分组
- **周**：按"该周周一日期"分组（周一起始）。周一 key = `addDays(dateStr, -((getUTCDay+6) % 7))`
- **月**：按 `YYYY-MM`（`date.slice(0,7)`）分组
- 每期聚合出：节数、总时长、平均 CPM、平均正确率；按期 key 升序排列

## 5. 聚合逻辑（纯函数 TDD）

- `period.ts`：
  - `weekKey(dateStr): string` — 该周周一日期
  - `monthKey(dateStr): string` — YYYY-MM
- `aggregate.ts`：
  - `interface PeriodStat { label: string; count: number; totalSec: number; avgCpm: number; avgAccuracy: number }`
  - `aggregate(sessions: SessionRow[], dim: 'day'|'week'|'month'): PeriodStat[]`
    - 按 dim 对应 key 分组（日=date、周=weekKey、月=monthKey）
    - 每组：`count`=节数；`totalSec`=Σ elapsedSec；`avgCpm`=`cpm(Σflips, Σsec)`（整期翻转/整期时长，复用 1a 的 `cpm`）；`avgAccuracy`=`Σcorrect / Σanswered`（answered 为 0 则 0）
    - 结果按 label 升序
- 纯函数、日期/数据参数注入，可测。

## 6. 图表组件（手绘 SVG）

- `LineChart({ values: number[]; labels: string[]; unit?: string })`：折线 + 简单坐标轴，自动按 values 极值缩放 Y。
- `BarChart({ values: number[]; labels: string[] })`：柱状。
- 纯展示组件（props 进、SVG 出），无数据获取。空数据显示占位。

## 7. 数据流与 UI

- `StatsPage`：挂载时 `db.sessions.toArray()` 读全部 → 按当前 dim `aggregate` → 渲染汇总卡片 + 三张图。
- 顶部三个按钮切日/周/月（`useState` dim）。
- 无 sessions 时显示"还没有训练记录，先去练一次吧"。

## 9. 测试策略

- **纯逻辑 TDD**：
  - `period`：weekKey（跨周/周一边界）、monthKey。
  - `aggregate`：按日/周/月分组、count/totalSec/avgCpm/avgAccuracy 计算、升序、空输入、answered=0 保护。
- **手动验证**：统计页三张图渲染、维度切换、空态、汇总卡片数字（练几轮后看）。

## 10. 默认决策（已与用户确认）

1. 手绘 SVG/CSS 图表，无第三方图表库
2. 展示：累计次数/时长卡片 + CPM 折线 + 正确率折线 + 次数柱状
3. 日/周/月三个维度都做
4. avgCpm 用整期 `cpm(Σflips, Σsec)`；avgAccuracy 用 `Σcorrect/Σanswered`
5. 周按周一起始分组
