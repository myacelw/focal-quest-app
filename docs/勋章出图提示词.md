# 30 勋章 AI 出图提示词（护眼小伙伴团）

给「变焦大冒险 / FocalQuest」勋章墙的 30 个勋章生成成套图标。统一画风 + 稀有度边框，替换现在的 emoji 占位。

## 怎么用

每张图 = **通用风格前缀** + **该勋章主体** + **稀有度边框后缀**，三段拼成一句喂给 GPT-4o / Gemini / Midjourney 等。

- 逐张生成，保持前缀不变 → 30 个风格统一、能成套。
- 若模型支持"参考上一张风格"，第一张定稿后让后续都参考它，一致性更好。

## 通用规格

- 输出 **1024×1024**（或 512×512）、**正方形**、**透明背景 PNG**。
- 角色**居中**，四周留白（给稀有度边框留空间）。
- App 内最终显示 96×96，所以细节别太碎。
- 全套统一：同一线条粗细、同一配色体系、同一光照。

## 通用风格前缀（Style Prefix，所有勋章共用）

```
Cute kawaii chibi mascot icon for a children's eye-training game, flat vector illustration with soft rounded shapes, bright cheerful colors, thick clean white outline, single centered character, transparent background, square 1:1 composition, game achievement badge, polished mobile game UI asset, adorable and friendly
```

## 稀有度边框后缀（Rarity Suffix，按勋章稀有度选一个）

- **铜 bronze**：`, enclosed in a simple round bronze medal ring`
- **银 silver**：`, enclosed in a shiny round silver medal ring with a soft glow`
- **金 gold**：`, enclosed in an ornate golden medal ring with radiating light rays`
- **彩虹 rainbow（传说）**：`, enclosed in a dazzling rainbow-gradient ring with sparkles and a glowing legendary aura`

## 出图后怎么接入（我来做）

1. 你把生成的 30 张图按 **`<id>.png`** 命名（如 `streak-7.png`、`cpm-15.png`），放到 `public/badges/`。
2. 告诉我放好了，我改 `BadgeCard.tsx`：把 emoji 换成 `<img src={`/badges/${def.id}.png`} width={96} height={96}>`，**emoji 作为图片加载失败时的兜底**（哪张没出图就还显示 emoji，可渐进替换）。
3. 图片文件较大，会像 vosk 模型一样加进 `.gitignore` 不入库（或按你意愿入库）。
