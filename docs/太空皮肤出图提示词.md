# 太空射击皮肤 AI 出图提示词

给 space skin 的 emoji 占位（🚀战机 / 🪨陨石 / 💥爆炸 / 💎彩蛋）升级成成套真图。星空背景用 CSS 已够好，可不出图。

## 规格

- **透明背景 PNG**，正方形，元素居中留白。
- 战机/陨石/彩蛋：512×512；爆炸：512×512（放射状）。
- 统一科幻卡通风、明亮、儿童友好，和勋章的"护眼小伙伴团"同一调性（圆润、厚描边、可爱）。

## 通用风格前缀（所有元素共用）

```
Cute cartoon sci-fi game asset for a children's space shooter, flat vector illustration, soft rounded shapes, bright vivid colors, thick clean outline, single centered object, transparent background, square composition, adorable kid-friendly, polished mobile game UI sprite
```

## 元素主体（拼在前缀后）

| 文件名 | 主体（英文 Subject） | 说明 |
|---|---|---|
| `ship.png` | a cute chubby little spaceship / starfighter pointing upward, glowing engine | 底部战机，机头朝上 |
| `asteroid.png` | a cute round grey space asteroid rock with small craters, friendly | 四方向陨石 |
| `explosion.png` | a vibrant cartoon explosion burst, orange-yellow star shape with sparks | 击中爆炸，放射状 |
| `treasure.png` | a cute glowing golden treasure chest with sparkles, slightly open | 彩蛋宝箱（替 💎）|
| `laser.png`（可选）| a bright cyan-white energy laser beam, vertical glowing bar | 激光（也可保持 CSS 渐变条）|

拼法示例（战机）：
`<风格前缀>, a cute chubby little spaceship pointing upward with a glowing engine`

## 出图后接入（我来做）

1. 图按上表文件名放到 `public/skins/space/`。
2. 告诉我放好了，我把 `SpaceStage.tsx` 里的 emoji 换成 `<img src="/skins/space/xxx.png">`：
   - 战机 🚀 → `ship.png`（底部）
   - 四方向陨石 🪨 → `asteroid.png`
   - 爆炸 💥 → `explosion.png`（命中动画里）
   - 彩蛋 💎 → `treasure.png`
   - 激光：可保持现在的 CSS 渐变条，或用 `laser.png`
3. emoji 作为图片加载失败的兜底，可一个个渐进替换。
4. 图片体积注意压缩（每张几十 KB 即可），放 `public/` 会打包。

## 备注

星空背景当前是 CSS 星点 + 深色渐变，效果已经不错、零资源，建议保持；若想更炫可出一张 `starfield.png` 平铺，但非必需。
