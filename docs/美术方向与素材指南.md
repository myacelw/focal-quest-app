---

## 1. 美术方向（修正）

- **目标观感：青少年向"真游戏质感"**。参照系是她玩过的真游戏（塞尔达旷野之息/王国之泪级别的风格化美术）——氛围、光影、材质感。
- **禁止**：低幼卡通（大头萌物、粗黑描边贴纸风、幼儿园配色）。
- **手段修正**：场景素材**优先用现成的高品质游戏素材包**（网上下载），而非 AI 生成卡通图。AI 生成仅用于补缺口，且提示词按"风格化写实/painterly"走，不再用 kawaii/chibi。
- 已有的勋章图（萌系）暂保留——勋章是收集品语境能容忍一些可爱；若她也嫌幼稚，再按本指南重做。
- 像素风（高质感像素艺术，如 Octopath/星露谷观感）是可接受的"真游戏"风格，且素材最丰富；手绘 painterly 也可。**最终以她挑中的素材包风格为准，一个场景内必须同一包/同一作者，不混搭。**

## 2. 素材源清单（免费/可商用）

| 站点 | 适用 | 许可要点 |
|---|---|---|
| [itch.io 免费素材区](https://itch.io/game-assets/free) | 主力。搜 tag：`dark-fantasy` / `temple` / `dungeon` / `sci-fi` / `parallax` | 逐包看许可，很多免费包可商用；下载页写明 |
| [OpenGameArt](https://opengameart.org) | fantasy/sci-fi 全类 | 每件标注 CC0/CC-BY/GPL，选 CC0 或 CC-BY（要署名）|
| [CraftPix Freebies](https://craftpix.net/freebies/) | 高质感 fantasy/RPG 免费区 | 免费区可商用，不可转售素材本身 |
| [NASA 图像库](https://images.nasa.gov) | **星际场景背景**：真实星云/深空照片 | 公版（public domain），太空场景的"零成本质感王牌" |
| [Game-icons.net](https://game-icons.net) | 图标（武器/道具/UI） | CC-BY 3.0，需页脚署名 |
| [Kenney](https://kenney.nl/assets) | UI 元素/音效 | 全 CC0；但画风扁平，只用于 UI 不用于场景 |

**国内访问**：OpenGameArt / CraftPix / NASA 一般可直连；itch.io 可能慢，耐心或换时段。

## 3. 挑选流程（分工）

1. **女儿挑**：打开上面链接（优先 itch.io 的 dark-fantasy / sci-fi tag 和 CraftPix freebies），看中哪个包记下链接。每个场景挑 1 个主包（背景+主要角色/物件同风格）。
2. **家长核**：确认该包许可允许使用（免费+可商用最好；CC-BY 记下作者名，我们在"关于"页署名）。下载 zip。
3. **接入**：解压出用到的图，按第 4 节规范放入项目，告诉 Claude 接入。

## 4. 接入规范

- 目录：`public/skins/<skinId>/`（如 `public/skins/space/`、`public/skins/shrine/`）。
- 命名：小写英文语义名，如 `bg.png`（背景）、`ship.png`、`guardian.png`、`explosion.png`、`rune-glow.png`。
- 格式：PNG（透明）或 JPG（背景）；单图 ≤ 300KB（大图先压缩）；背景可用 1024–2048 宽。
- 序列帧动画：横向拼条命名 `xxx-strip8.png`（8 帧），接入时按帧宽切。
- 大素材包原始 zip 不入 git（放 `public/skins/` 的用图入 git 即可，因已挑选且压缩过）。
- 每个皮肤目录放一个 `CREDITS.md`：来源链接 + 作者 + 许可（合规留档，商业化时直接可查）。

## 5. 各场景素材购物清单

### 神庙勇者（塞尔达风，优先）
- 神庙内景背景（暗石材质 + 青绿发光纹路，或 parallax 遗迹背景）
- 守护者/魔法核心（悬浮眼球/机械核心，带发光感）
- 光之箭/剑光特效（序列帧更好）
- 光之精灵（发光小球/蝴蝶状）
- 关键词：`ancient temple`, `magic shrine`, `glowing runes`, `guardian eye`, `light effects`, `parallax ruins`

### 星际歼灭战 V2
- 背景：**NASA 真实星云图**（如猎户座星云、创生之柱——公版，直接下载即质感天花板）
- 战机 sprite（俯视/侧视，风格化写实）
- 敌舰/护盾（能量罩感）
- 爆炸序列帧、激光束
- 关键词：`spaceship sprite pack`, `space shooter assets`, `explosion sprite sheet`

### 深海寻宝 / 符文之门 / 驯龙骑士（后续）
- 等前两个场景跑通流程后，同法炮制。

## 6. 技术接入方式（Claude 侧）

- Stage 组件里 emoji/CSS 占位 → `<img>` / `background-image` 引用 `public/skins/<id>/` 下的图。
- 图片加载失败时回退占位（渐进替换，不阻塞）。
- 序列帧用 CSS `steps()` 动画播放。
- 光效（发光、辉光）继续用 CSS `box-shadow`/`filter: drop-shadow` 叠加在素材上，省素材且质感好。
